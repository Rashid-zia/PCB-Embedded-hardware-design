This project focuses on the design and PCB layout of a 3W audio amplifier, developed as part of my bachelor studies in Applied Informatics in Industrial Automation. ## Objectives
- Design a functional 3W audio amplifier circuit
- Create schematic and PCB layout using EAGLE
- Design custom component footprints
- Optimize routing, grounding, and power distribution
- Ensure manufacturability and compliance with PCB layout standards
- Prepare fabrication-ready design files
